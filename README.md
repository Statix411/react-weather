## Простое приложение погоды на React.js

Данные о погоде берутся с сайта OpenWeatherMap.

####Выводятся данные о:<br/>
  *Городе
  *Температуре
  *Восходе
  *Заходе
  *Давлении
  *Ветре


####Дизайн минималистический Flat.


###Установка

#### `npm i`

###Запуск

#### `npm start`

Приложение запускается на [http://localhost:3000](http://localhost:3000)
